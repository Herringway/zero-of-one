# zero-of-one
(Yet another) Markov chain based IRC reply bot.

## Dependencies
- POSIX compliant OS.
- C compiler (with C99 support).
- CMake.

## Quick setup guide
```
$ git clone https://github.com/nsensfel/zero-of-one.git
$ cd zero-of-one
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./zero_of_one -h
```

## Main Objectives
- POSIX compliance.
- Paranoia levels of error checking.
- Low RAM usage.
- Giggles.
